What We Do

GOTO ANYTHING

Use Goto Anything to open files with only a few keystrokes, and instantly jump to symbols, lines or words.

MULTIPLE SELECTIONS

Multiple selections allow you to interactively change many lines at once, rename variables with ease, and manipulate files faster than ever.

POWERFUL API AND PACKAGE ECOSYSTEM

Sublime Text has a powerful, Python API that allows plugins to augment built-in functionality.

SPLIT EDITING

Get the most out of your wide screen monitor with split editing support. Edit files side by side, or edit two locations in the one file.

PERFORMANCE

Sublime Text is built from custom components, providing for unmatched responsiveness.

CROSS PLATFORM

Sublime Text is available for Mac, Windows and Linux.
